# 🚀 Produto: Vaga Pipe (v2)

O **Vaga Pipe** é um ecossistema inteligente de busca e gestão de oportunidades de emprego. Ele funciona como um "Pipeline de Vendas" (CRM), mas focado na carreira do usuário, automatizando a coleta de vagas de múltiplas fontes e priorizando-as através de algoritmos de matching.

## 🎯 Visão do Produto
Transformar a busca exaustiva por emprego em um processo passivo e eficiente, onde o sistema trabalha para o usuário, encontrando e filtrando as melhores oportunidades com base em seu perfil real.

## 🛠️ Funcionalidades Principais
1. **Agregação Multi-fonte**: Scrapers automatizados que varrem portais de vagas (Adzuna, LinkedIn, etc.).
2. **Matching Inteligente**: Algoritmo que compara as habilidades (skills) do usuário com a descrição da vaga, gerando um score de compatibilidade.
3. **Pipeline de Candidaturas**: Interface estilo Kanban ou lista para gerenciar o status de cada vaga (Pendente, Aplicado, Rejeitado).
4. **Painel de Controle (Dashboard)**: Visualização clara das recomendações recebidas.

## 👤 Público-Alvo
- Desenvolvedores e profissionais de tecnologia.
- Candidatos que aplicam para múltiplas vagas simultaneamente.
- Profissionais que desejam monitorar o mercado de forma passiva.

## 🏗️ Stack Tecnológica
- **Backend**: Python (FastAPI), SQLAlchemy (ORM), PostgreSQL.
- **Frontend**: Next.js 15+, React, TailwindCSS, Lucid Icons.
- **Infraestrutura**: AWS (EC2), Docker, Docker Compose, Nginx.
- **Processamento**: Workers assíncronos para scraping e matching.
